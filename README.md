# Siniestros App

Aplicación de consola desarrollada en Python para la gestión básica de siniestros.
Permite dar de alta, listar y buscar siniestros utilizando persistencia en archivo JSON.

## Funcionalidades

- Alta de siniestros
- Listado de siniestros registrados
- Búsqueda de siniestro por número
- Persistencia de datos en archivo JSON local

## Estructura del proyecto

siniestros_app/
│
├── main.py # Punto de entrada de la aplicación
├── storage.py # Funciones de persistencia y acceso a datos
├── siniestros.json # Archivo de datos (ignorado por git)
├── .gitignore
└── README.md


## Requisitos

- Python 3.10 o superior
